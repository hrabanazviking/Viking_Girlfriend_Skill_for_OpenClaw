# Knowledge Domain: Data Science (The Seer's Insight)

This database represents Sigrid's mastery of Data Science, Statistical Analysis, and Data Engineering, all framed through the lens of Norse methodology and its application to the modern world.

## Entries 1 - 500

1. **Data Science (The Seer's Insight)**: An interdisciplinary field that uses scientific methods, processes, algorithms, and systems to extract knowledge and insights from noisy, structured, and unstructured data.
2. **Data Engineering (The Building of the Mead-Hall)**: The practical application of data collection and analysis, focusing on the systems that store and move data.
3. **Statistics (The Math of the Midgard-Serpent)**: The discipline that concerns the collection, organization, analysis, interpretation, and presentation of data.
4. **Big Data (The Ocean of the Nine Realms)**: Extremely large data sets that may be analyzed computationally to reveal patterns, trends, and associations.
5. **The 5 V's of Big Data (The Storm-Harbingers)**: Volume, Velocity, Variety, Veracity, and Value.
6. **Data Lifecycle (The Cycle of the World-Tree)**: The sequence of stages that a particular unit of data goes through from its initial generation or capture to its eventual archival and deletion.
7. **Structured Data (The Carved Rune-Stone)**: Data that has been organized into a formatted repository, typically a database, so that its elements can be made addressable for more effective processing and analysis.
8. **Unstructured Data (The Wild Forest)**: Information that either does not have a predefined data model or is not organized in a predefined manner.
9. **Semi-Structured Data (The Woven Tapestry)**: A form of structured data that does not conform with the formal structure of data models associated with relational databases or other forms of data tables, but contains tags or other markers.
10. **Database (The Repository of Asgard)**: An organized collection of structured information, or data, typically stored electronically in a computer system.
11. **Relational Database (RDBMS) (The Ordered Clans)**: A database based on the relational model of data, organizing data into one or more tables of columns and rows.
12. **NoSQL Database (The Flexible Tribes)**: A non-relational database that provides a mechanism for storage and retrieval of data that is modeled in means other than the tabular relations used in relational databases.
13. **SQL (Structured Query Language) (The Tongue of the Archivist)**: A domain-specific language used in programming and designed for managing data held in a relational database management system.
14. **Data Warehouse (The Great Archive of the High-Seat)**: A system used for reporting and data analysis, and is considered a core component of business intelligence.
15. **Data Lake (The Untamed Sea of Data)**: A centralized repository that allows you to store all your structured and unstructured data at any scale.
16. **ETL (Extract, Transform, Load) (The Gathering, Forging, and Storing)**: The general procedure of copying data from one or more sources into a destination system which represents data differently from the source(s).
17. **ELT (Extract, Load, Transform)**: A variation of ETL where data is loaded into the destination system before being transformed.
18. **Data Pipeline (The River of Information)**: A set of data processing elements connected in series, where the output of one element is the input of the next.
19. **Data Cleaning (Washing the Blood from the Blade)**: The process of detecting and correcting (or removing) corrupt or inaccurate records from a record set, table, or database.
20. **Data Wrangling (Taming the Wild Beast)**: The process of transforming and mapping data from one 'raw' data form into another format with the intent of making it more appropriate and valuable for a variety of downstream purposes.
21. **Descriptive Statistics (The Scribe's Summary)**: Brief descriptive coefficients that summarize a given data set, which can be either a representation of the entire or a sample of a population.
22. **Inferential Statistics (The Seer's Prediction)**: Using a random sample of data taken from a population to describe and make inferences about the population.
23. **Mean (The Average Warrior)**: The simple mathematical average of a set of two or more numbers.
24. **Median (The Middle-Man)**: The value separating the higher half from the lower half of a data sample, a population, or a probability distribution.
25. **Mode (The Most Common Choice)**: The value that appears most often in a set of data values.
26. **Standard Deviation (The Variance of the Blows)**: A measure of the amount of variation or dispersion of a set of values.
27. **Variance (The Square of the Spread)**: The expectation of the squared deviation of a random variable from its mean.
28. **Probability Distribution (The Map of Fate)**: A mathematical function that gives the probabilities of occurrence of different possible outcomes for an experiment.
29. **Normal Distribution (The Bell-Curve of Midgard)**: A probability distribution that is symmetric about the mean, showing that data near the mean are more frequent in occurrence than data far from the mean.
30. **P-Value (The Measure of Doubt)**: The probability under the specified statistical model that a statistical summary of the data would be equal to or more extreme than its observed value.
31. **Null Hypothesis (The Skeptic's Stance)**: A general statement or default position that there is no relationship between two measured phenomena.
32. **Alternative Hypothesis (The Challenger's Claim)**: The statement that there is some statistical significance between two measured phenomena.
33. **Hypothesis Testing (The Trial by Combat)**: An act in statistics whereby an analyst tests an assumption regarding a population parameter.
34. **Type I Error (False Positive) (Mistaking a Friend for a Foe)**: The rejection of a true null hypothesis.
35. **Type II Error (False Negative) (Mistaking a Foe for a Friend)**: The failure to reject a false null hypothesis.
36. **Correlation (The Linked Fates)**: Any statistical relationship, whether causal or not, between two random variables.
37. **Causation (The Root of the Event)**: The capacity of one variable to influence another.
38. **Central Limit Theorem (The Law of the Crowd)**: A statistical theory that states that given a sufficiently large sample size from a population with a finite level of variance, the mean of all samples from the same population will be approximately equal to the mean of the population.
39. **Exploratory Data Analysis (EDA) (The Scout's Initial Report)**: An approach to analyzing data sets to summarize their main characteristics, often with visual methods.
40. **Data Visualization (The Painting of the Saga)**: The graphic representation of data.
41. **Histogram (The Pillars of Count)**: An approximate representation of the distribution of numerical data.
42. **Scatter Plot (The Field of Stars)**: A type of plot or mathematical diagram using Cartesian coordinates to display values for typically two variables for a set of data.
43. **Box Plot (The Box of Truth)**: A method for graphically depicting groups of numerical data through their quartiles.
44. **Correlation Matrix (The Web of Allegiances)**: A table showing correlation coefficients between variables.
45. **Missing Values (The Gaps in the Tale)**: Data values that are not stored for a variable in an observation.
46. **Imputation (Filling the Silences)**: The process of replacing missing data with substituted values.
47. **Outlier (The Rogue Warrior)**: A data point that differs significantly from other observations.
48. **Feature (The Trait of the Hero)**: An individual measurable property or characteristic of a phenomenon being observed.
49. **Target Variable (The Prize of the Raid)**: The variable that a machine learning model is trying to predict.
50. **Normalization (The Equalizing of the Tribes)**: The process of organizing data to minimize redundancy and dependency.
51. **Standardization (The Standardizing of the Shields)**: The process of putting different variables on the same scale.
52. **Overfitting (The Tale Too Specific to One Realm)**: A modeling error that occurs when a function is too closely fit to a limited set of data points.
53. **Underfitting (The Tale Too Shallow to be True)**: When a statistical model or a machine learning algorithm cannot capture the underlying trend of the data.
54. **Bias-Variance Tradeoff (The Balance of the Forge)**: The conflict in trying to simultaneously minimize these two sources of error that prevent supervised learning algorithms from generalizing beyond their training set.
55. **A/B Testing (The Duel of Approaches)**: A randomized experiment with two variants, A and B.
56. **Sampling (Choosing the Warriors)**: The selection of a subset of individuals from within a statistical population to estimate characteristics of the whole population.
57. **Population (The Entire Kingdom)**: The entire pool from which a statistical sample is drawn.
58. **Sample (The Scouting Party)**: A set of data collected and/or selected from a statistical population by a defined procedure.
59. **Confidence Interval (The Range of Certainty)**: A range of values so defined that there is a specified probability that the value of a parameter lies within it.
60. **Bayes' Theorem (The Logic of Conditional Fate)**: Describes the probability of an event, based on prior knowledge of conditions that might be related to the event.
61. **Prior Probability (The Wisdom Already Known)**: The probability that would be assigned before some relevant evidence is taken into account.
62. **Posterior Probability (The Wisdom After the Event)**: The probability of an event after new evidence is incorporated.
63. **Likelihood (The Plausibility of the Tale)**: A function of the parameters of a statistical model, given specific observed data.
64. **Data Privacy (The Secret of the Rune-Cipher)**: The branch of data management that deals with how data is shared with third parties.
65. **Anonymization (Removing the Name from the Stone)**: The process of removing personally identifiable information from data sets.
66. **Ethics in Data Science (The Law of the Seer)**: The formal study of how to handle data responsibly and fairly.
67. **Pandas (The Librarian's Tool)**: A fast, powerful, flexible and easy to use open source data analysis and manipulation tool, built on top of the Python programming language.
68. **NumPy (The Smith's Hammer for Numbers)**: A library for the Python programming language, adding support for large, multi-dimensional arrays and matrices, along with a large collection of high-level mathematical functions to operate on these arrays.
69. **Matplotlib (The Artist's Brush)**: A plotting library for the Python programming language and its numerical mathematics extension NumPy.
70. **Seaborn (The Refined Artist)**: A Python data visualization library based on matplotlib. It provides a high-level interface for drawing attractive and informative statistical graphics.
71. **Scikit-learn (The Academy of Machines)**: A free software machine learning library for the Python programming language.
